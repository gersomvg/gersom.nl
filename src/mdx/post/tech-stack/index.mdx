Alternatively, you can check out the [package.json](https://github.com/gersomvg/gersom.nl/blob/master/package.json).

Libraries (most important):

- [TypeScript](https://www.typescriptlang.org) - Typed superset of JavaScript
- [React](https://reactjs.org) - Library for building user interfaces
- [GatsbyJS](https://www.gatsbyjs.org) - Static site generator
- [MDX](https://mdxjs.com) - Markdown mixed with JSX
- [Prism](https://prismjs.com) - Syntax highlighting for code blocks
- [styled-components](https://www.styled-components.com) - CSS-in-JS

Hosting:

- [Netlify](https://www.netlify.com) - Static & Serverless

Other:

- [VSCode](https://code.visualstudio.com) - IDE
- [Github](https://github.com/gersomvg/gersom.nl/) - Git Repository
